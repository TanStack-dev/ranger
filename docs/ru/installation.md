# Установка

Вы можете установить TanStack Ranger с помощью любого менеджера пакетов [NPM](https://npmjs.com).

В зависимости от выбранного фреймворка установите один из следующих пакетов:

- [React](./adapters/react-ranger)
- Solid (скоро будет!)
- Vue (скоро будет!)
- Svelte (скоро будет!)
- Angular (скоро будет!)
