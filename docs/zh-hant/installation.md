# 安裝 (Installation)

您可以透過任何 [NPM](https://npmjs.com) 套件管理器來安裝 TanStack Ranger。

根據您選擇的框架，安裝以下其中一個套件：

- [React](./adapters/react-ranger)
- Solid (即將推出！)
- Vue (即將推出！)
- Svelte (即將推出！)
- Angular (即將推出！)
